# herdr: Package for Animal Greenhouse Gas Emissions and Land Use

This package provides tools for calculating and analyzing greenhouse gas
emissions and land use metrics for animal systems.

Description: Provides a comprehensive set of tools for calculating
livestock greenhouse gas emissions based on Spanish national standards.
The package includes functions to process animal census data for cattle,
sheep, and goats, integrating specific emission factors and rate
parameters. It facilitates the estimation of direct and indirect
emissions, such as methane and nitrous oxide, following standardized
methodologies for environmental reporting.
