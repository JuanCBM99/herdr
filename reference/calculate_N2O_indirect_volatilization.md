# Calculate indirect N₂O emissions from volatilization (Refactored)

Computes indirect N₂O emissions derived from volatilization of excreted
nitrogen (IPCC Eq 10.26 and 10.28).

## Usage

``` r
calculate_N2O_indirect_volatilization(saveoutput = TRUE)
```

## Arguments

- saveoutput:

  If TRUE (default) the results are saved in the output folder.
