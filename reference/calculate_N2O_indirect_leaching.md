# Calculate indirect N₂O emissions from leaching (Refactored)

Computes indirect N₂O emissions derived from nitrogen leaching (IPCC Eq
10.27 and 10.29).

## Usage

``` r
calculate_N2O_indirect_leaching(saveoutput = TRUE)
```

## Arguments

- saveoutput:

  If TRUE (default) the results are saved in the output folder.
